    <h1>Java Scientific Calculator GUI Project</h1>

    <img src="demo.png" alt="Calculator Demo">

    <h2>Overview</h2>
    <p>This Java project is a feature-rich scientific calculator with a Graphical User Interface (GUI). It offers an intuitive interface for performing complex mathematical calculations conveniently.</p>

    <h2>Features</h2>
    <ul>
        <li>Standard and Scientific Functions: Includes basic arithmetic operations along with a wide range of scientific functions such as trigonometric, logarithmic, and exponential functions.</li>
        <li>Graph Plotting: Plot graphs of various functions to visualize mathematical relationships.</li>
        <li>Memory Functionality: Store and recall values using memory buttons.</li>
        <li>History: View a history of previous calculations for reference.</li>
        <li>Customization: Customize the interface with themes and settings.</li>
    </ul>

    <h2>Getting Started</h2>
    <ol>
        <li><strong>Prerequisites:</strong> Ensure you have Java installed on your system.</li>
        <li><strong>Clone the Repository:</strong> <code>git clone https://github.com/yourusername/java-scientific-calculator.git</code></li>
        <li><strong>Build and Run:</strong> Open the project in your favorite Java IDE and run the <code>CalculatorGUI.java</code> file.</li>
    </ol>

    <h2>Usage</h2>
    <ul>
        <li><strong>Basic Operations:</strong> Perform addition, subtraction, multiplication, and division using the standard buttons.</li>
        <li><strong>Scientific Functions:</strong> Access trigonometric, logarithmic, and exponential functions from the scientific panel.</li>
        <li><strong>Graph Plotting:</strong> Enter a function and click the "Plot" button to visualize it on the graph.</li>
        <li><strong>Memory Operations:</strong> Store values in memory using M+, recall them using MR, and clear memory using MC.</li>
    </ul>
